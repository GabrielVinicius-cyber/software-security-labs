# Roadmap — Lab-01-Flask

## Missão 1 — Configurar ambiente ✅
Git instalado e configurado, Python + venv validados.

## Missão 2 — Estrutura do projeto + documentação inicial ✅
Estrutura de pastas criada, README.md e .gitignore escritos.

## Missão 3 — Criar aplicação Flask mínima
Uma rota "hello world", rodando localmente, com boas práticas
de organização de código (sem lógica de negócio ainda).

## Missão 4 — Escrever primeiro teste
Um teste simples validando que a rota responde corretamente.

## Missão 5 — Criar script utilitário
Um script em `scripts/` para automatizar algo do ambiente
(ex: criar o venv e instalar dependências de uma vez).

## Missão 6 — Fechar CHANGELOG + revisão geral
Documentar tudo no CHANGELOG.md e revisar antes de considerar
o Lab-01 encerrado.
