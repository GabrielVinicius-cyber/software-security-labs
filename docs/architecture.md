# Arquitetura — Lab-01-Flask

## Visão geral / Overview

🇧🇷 **PT-BR:** Laboratório prático para aprender os fundamentos de uma
aplicação Flask em Python, com foco em versionamento e organização de
projeto — base para os labs de segurança seguintes.

🇺🇸 **EN:** Practical lab to learn the fundamentals of a Flask
application in Python, focused on version control and project
organization — the foundation for the security labs that follow.

## Decisões técnicas / Technical decisions

### Por que Python + Flask? / Why Python + Flask?

🇧🇷 **PT-BR:** Escolhi Python por ser uma das linguagens mais fortes em
cibersegurança, e quero me aprofundar até um nível avançado. Flask serve
para entender o que acontece por trás de uma aplicação web (login, rotas,
etc.), e onde procurar e evitar falhas.

🇺🇸 **EN:** I chose Python for being one of the strongest languages in
cybersecurity, aiming to reach an advanced level. Flask helps me
understand what happens behind a web application (login, routes, etc.),
and where to look for and prevent vulnerabilities.

### Por que Git desde o início? / Why Git from the start?

🇧🇷 **PT-BR:** Além de construir portfólio, o Git permite rastrear cada
decisão tomada, reverter erros com segurança e criar um histórico de
aprendizado — cada commit é uma prova de progresso.

🇺🇸 **EN:** Besides building a portfolio, Git allows tracking every
decision made, safely reverting mistakes, and creating a learning
history — each commit is proof of progress.

### Por que essa estrutura de pastas? / Why this folder structure?

🇧🇷 **PT-BR:** Para manter os laboratórios bem estruturados e organizados
para estudo, permitindo revisar o conteúdo conforme a dúvida surgir, e
tornando mais fácil para quem está vendo o repositório entender do que
se trata.

🇺🇸 **EN:** To keep the labs well-structured and organized for study,
allowing content to be revisited whenever a question arises, and making
it easier for anyone viewing the repository to understand what it is
about.

## Limitações atuais / Current limitations

🇺🇸 **EN:** No complex application has been implemented yet.
